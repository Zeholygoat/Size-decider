# Euphoria Size Match 💋

A playful neon-themed personality quiz website inspired by the visual vibes of Euphoria.

## Features
- Interactive vibe quiz
- Stylish glassmorphism UI
- Neon pink/purple gradients
- Animated results
- Fun personality-based sizing system

## Files
- index.html
- style.css
- script.js

## Run
Simply open `index.html` in your browser.
